# 🌐 Web Development Mini Projects

A collection of small web development projects I built during my learning journey.<br>
This repository is mainly focused on improving my understanding of core frontend concepts, JavaScript logic, APIs, DOM manipulation, and browser features.<br><br>

## 🚀 Projects Included

### 🎮 Tic Tac Toe
A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.<br>
Focused on DOM manipulation and game logic.

<br>

### 💱 Currency Converter
A currency converter using a real-time API.<br>
Helped me learn API fetching, async JavaScript, and dynamic UI updates.

<br>

### 📧 Email & Password Validator
A basic validator project for checking email and password formats.<br>
Focused on form handling and regular expressions.

<br>

### 🎨 Image Editor
A beginner-friendly image editor using the Canvas API.<br>
Includes live filters, reset functionality, and image download support.<br>
This project helped me understand canvas rendering and image manipulation in JavaScript.

<br>
