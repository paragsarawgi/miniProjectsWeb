# 🎨 Image Editor

A basic browser-based Image Editor built using **HTML, CSS, and JavaScript**.<br>
This project was created mainly for learning and experimenting with the **Canvas API** and image manipulation in JavaScript.<br><br>

## 🚀 Features

- 📂 Upload images
- 🖼️ Display images on canvas
- 🎚️ Live image editing using sliders
- ☀️ Brightness adjustment
- 🌈 Contrast, Saturation, Blur, Sepia, Invert, Hue Rotate, Opacity controls
- 🔄 Reset functionality
- ⬇️ Download edited image
- ⚡ Real-time canvas rendering

<br>

## 🧠 What I Learned

This project helped me understand:

- HTML Canvas API
- Image rendering on canvas
- JavaScript event handling
- Live filter updates
- Downloading files using JavaScript
- Managing editor state and redraw logic

<br>

## ⚠️ Note

This is a beginner-level project and not a professional image editor.<br>
The main goal was learning how image processing works inside the browser using JavaScript and Canvas.
