# 💱 Currency Converter

A simple and responsive Currency Converter web app built using HTML, CSS, and JavaScript.<br>
This project focuses on clean UI design, responsive layouts, and real-time currency conversion using APIs.<br><br>

## 🚀 Features

- Convert currencies instantly<br>
- Responsive design for mobile, tablet, and desktop<br>
- Clean and modern UI<br>
- Real-time exchange rates using API<br>
- Beginner-friendly project structure<br><br>

## 🛠️ Technologies Used

- HTML5<br>
- CSS3<br>
- JavaScript<br>
- Currency Exchange API<br><br>
