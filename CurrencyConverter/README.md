# 💱 Currency Converter

A simple and responsive Currency Converter web app built using HTML, CSS, and JavaScript.<br>
This project focuses on clean UI design, responsive layouts, and real-time currency conversion using APIs.<br><br>

# AI usage

Leveraged AI assistants, including ChatGPT and Claude, to accelerate my mastery of asynchronous JavaScript and API integrations, simultaneously utilizing them to refine and optimize my codebase.<br><br>

## 🚀 Features

- Convert currencies instantly<br>
- Responsive design for mobile, tablet, and desktop<br>
- Clean and modern UI<br>
- Real-time exchange rates using API<br>
- Beginner-friendly project structure<br><br>

## 🛠️ Technologies Used

- HTML5<br>
- CSS3<br>
- JavaScript<br>
- Currency Exchange API<br><br>

## 📌 Status

Currently functional and responsive, with plans for additional features such as improving the UI specialy the dropdown menu and adding more currencies in the future.
