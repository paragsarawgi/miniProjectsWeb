# Tic Tac Toe Game

A simple Tic Tac Toe game built using **HTML, CSS, and JavaScript**.<br>
This project was created during my web development learning journey to practice DOM manipulation, event handling, and game logic.

<br>

## 🚀 Features

- 🎮 Two-player gameplay
- ✨ Interactive UI
- 🧠 Win and draw detection
- 🔄 Reset / New Game functionality
- ⚡ Instant result updates

<br>

## 🧠 What I Learned

- DOM Manipulation
- JavaScript event handling
- Game state management
- Conditional logic
- Dynamic UI updates

<br>

## ⚠️ Note

This is a beginner-level learning project built for practice and understanding core JavaScript concepts.

<br>
